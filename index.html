Projects

Project 1 - Hogwarts Invaders

Hogwarts Invaders is a grid-based game inspired by Space Invaders using HTML, CSS and ‘Vanilla’ JavaScript. This was a 1-week solo project. The aim was to translate the key aspects of Harry Potter into the game, and so I focused heavily on animations, using set intervals and timeouts, and also the use of sound effects to create a sense of suspense and intensity around the game.

I would like in the future for the player to have the ability to block spells and if spells collide they are then removed from the grid.

Project 2 - JJDB

JJDB is a front-end web application built using React and a public API, allowing users to search for and compare their favourite movies. This was a paired project to be completed in 48-hours using a public API called The Movie Database API. This API gave us all the data to provide the user with the most relevant information when comparing two movies.

In the future Joao and I would like to work on the design of the project a bit more and also add movie trailers.

Project 3 - Booked-Up

Booked Up is a full-stack desktop web application with a React.js front end, a Node.js - Express back end, and a MongoDB database, allowing book lovers to search through thousands of books, create their own library and discuss these books in virtual book clubs. This was a group project with a time-frame of over a week and we used two external APIs - The Google Books API and NY Times API.

In the future we are looking to improve on the design of the project as a heavy focus was on the functionality.

Project 4 - Birdie Time

Birdie Time is a full-stack desktop web application with a React.js front end, a Django-Python back end, and a PostgreSQL database, looking to take avid golfers on a journey to playing the top 50 golf courses in the UK. This was an individual project completed in one week.

In the future I am looking to add an online instructor so that your swing can get the best tips and tricks, and secondly I would like to create virtual golf clubs where you can discuss courses with other users, compare scores and book a tee-time together.
